# Project Overview

Faisal Store is a responsive and user-friendly e-commerce website designed to deliver a modern online shopping experience.

 # The project focuses on:

- Clean and modern UI/UX
- Fully responsive design
- Smooth navigation experience
- Product browsing and filtering
- Shopping cart functionality
- Mobile-friendly interface

# Features

- Home Page
- Products Listing Page
- Search Functionality
- Shopping Cart
- Wishlist Section
- Responsive Design
- Dark Theme UI
- Category Filtering
- Fast Loading Performance

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Git & GitHub
- GitHub Pages



 Installation & Setup

Clone the repository:

git clone https://github.com/Faisal-laghari/MAIN-CAPSTONE_Complete_E-commerce_Website.git

Open the project folder:

cd MAIN-CAPSTONE_Complete_E-commerce_Website

Run the project by opening "index.html" in your browser.
